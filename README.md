# Dev Branch – Integration

The `dev` branch is the **integration branch** where new features are merged and tested before release.

## Key Highlights
- Merged changes from `feature` branch.
- Resolved merge conflicts in `demo.txt`.
- Pushed updated `dev` branch to remote.

## Purpose
This branch acts as a **staging ground** for new features.  
Once stable, changes are merged into `main` for release.
